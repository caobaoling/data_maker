# Redis Web管理工具使用说明

## 功能特性

### 1. 连接与监控
- ✅ 自动连接到默认Redis服务器 (172.16.70.21:6379)
- ✅ 实时显示统计信息：
  - 总键数
  - 内存使用量
  - Redis版本
  - 在线客户端数

### 2. 键搜索
- ✅ 支持模式匹配搜索（通配符）
- ✅ 常用模式示例：
  - `user:*` - 搜索所有以 user: 开头的键
  - `*cache*` - 搜索包含 cache 的键
  - `appoint_*` - 搜索预约相关的键
  - `*:123456` - 搜索以 123456 结尾的键

### 3. 键管理

#### 查看键详情
- 点击键名或"查看"按钮
- 支持的数据类型：
  - **String**: 显示文本内容
  - **Hash**: 表格形式展示字段和值
  - **List**: 表格形式展示索引和元素
  - **Set**: 表格形式展示成员
  - **ZSet**: 表格形式展示成员和分数
- 显示TTL（过期时间）：
  - 永久：-1
  - 已过期：-2
  - 剩余秒数：具体数字

#### 删除操作

**单个删除**
1. 点击键行的"删除"按钮
2. 二次确认后删除

**批量删除**
1. 勾选多个键（支持跨页选择）
2. 点击顶部"批量删除"按钮
3. 确认后批量删除

**按模式删除（⚠️ 危险操作）**
- **特点**：参考 `delete_redis.py` 的安全机制
- **流程**：
  1. 输入匹配模式（如：`test:*`）
  2. 点击"按模式删除"按钮
  3. **第一步 - 预览**：
     - 系统自动扫描匹配的键（dry_run=true）
     - 显示将要删除的键列表（前100个）
     - 显示匹配总数
  4. **第二步 - 确认**：
     - 仔细检查预览列表
     - 点击"确认删除"执行真正删除（dry_run=false）
     - 或点击"取消"放弃操作

### 4. 安全特性（参考 delete_redis.py）

✅ **两阶段删除流程**
- 第一阶段：dry_run=true，只查询不删除
- 第二阶段：用户确认后，dry_run=false，实际删除

✅ **预览确认机制**
- 显示匹配总数
- 显示前100个键名供检查
- 提示是否还有更多未显示的键

✅ **危险操作警告**
- 红色警告框提示不可恢复
- 显著的视觉提示

✅ **批量处理**
- 后端使用批次处理（batch_size=1000）
- 避免阻塞Redis服务器

## 操作示例

### 示例1：删除测试用户的预约缓存
```
1. 输入模式: appoint_multi_redis_*58153803*
2. 点击"按模式删除"
3. 查看预览列表，确认是该用户的缓存
4. 点击"确认删除 XX 个键"
```

### 示例2：清理过期的临时数据
```
1. 输入模式: temp:*
2. 点击"按模式删除"
3. 预览列表中检查键名
4. 确认后删除
```

### 示例3：查看Hash类型的数据
```
1. 搜索: user_info:123456
2. 点击"查看"
3. 以表格形式查看所有字段和值
```

## 与 delete_redis.py 的对比

### 相同点
- ✅ 支持 dry_run 预览模式
- ✅ 二次确认机制
- ✅ 批量处理避免阻塞
- ✅ 详细的日志记录

### 优势
- ✅ 可视化界面，更直观
- ✅ 支持键详情查看（多种数据类型）
- ✅ 支持多选批量删除
- ✅ 实时统计信息监控
- ✅ 无需修改脚本参数

## 注意事项

⚠️ **按模式删除是高危操作**
- 会删除所有匹配的键
- 删除操作不可恢复
- 请务必仔细检查预览列表

⚠️ **通配符使用**
- `*` 匹配任意字符（包括空）
- 避免使用单独的 `*`（会匹配所有键）
- 建议使用具体的前缀或后缀

⚠️ **生产环境使用**
- 建议先在测试环境验证
- 重要数据操作前备份
- 限制访问权限

## 技术架构

### 后端 (Flask)
- 路径：`backend/api/redis.py`
- 使用 `common/delete_redis.py` 的 `RedisKeyDeleter` 类
- 支持 dry_run 参数

### 前端 (Vue 3 + Element Plus)
- 路径：`frontend/src/views/redis/RedisTool.vue`
- 两阶段删除确认流程
- 响应式设计

### API接口
- `POST /api/redis/search` - 搜索键
- `POST /api/redis/key/info` - 获取键信息
- `POST /api/redis/key/delete` - 批量删除
- `POST /api/redis/key/delete_by_pattern` - 按模式删除（支持dry_run）
- `POST /api/redis/stats` - 获取统计信息
